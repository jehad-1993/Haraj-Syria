# 🚀 تحديث GitHub Repository

## إذا كان GitHub Desktop مثبت:
1. افتح GitHub Desktop
2. اختر repository "haraj-syria" 
3. استبدل ملف index.html بالنسخة الجديدة
4. اكتب commit message: "Fix navigation and language issues"
5. اضغط "Commit to main"
6. اضغط "Push origin"

## أو استخدم Command Line:
```bash
# إذا كان المشروع موجود محلياً
cd /path/to/haraj-syria
git add index.html
git commit -m "Fix navigation and language switching issues"
git push origin main
```

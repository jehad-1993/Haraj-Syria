The primary objective is to develop Haraj Syria, a classifieds website tailored for the Syrian market, inspired by the Haraj platform.
Core requirements include:
1.  **User Interface:** A simple, user-friendly interface with full Arabic language support (RTL), including an Arabic/English language toggle (changed from عر to ع), and enhanced UI elements like improved search bar hints and prominent language selection.
2.  **Classified Categories:** Multiple primary categories (cars, real estate, electronics, jobs) with comprehensive subcategories (e.g., سيارات سيدان for cars, specific car models linked to brands, years 2010-2050). All dropdowns (countries, cities, car details) should include an أخرى (Other) option. Categories on the homepage should display ad counts.
3.  **Ad Posting & Viewing:** Free and paid ad posting options, individual ad pages with photos, descriptions, prices, and locations. A robust image upload system.
4.  **User Management:** User registration requiring email, phone number, and name (no OTP initially). The registration form order should be Country, City, Phone, with the country code displayed next to the phone field. Password requirements should be visible. Login functionality and a Forgot Password feature via email/SMS link.
5.  **Geographic System:** Dropdowns for all Arab countries and their respective cities.
6.  **Search & Filtering:** An internal search engine with advanced filtering capabilities.
7.  **Contact Options:** Direct contact via WhatsApp or phone number.
8.  **Admin Panel:** Functionality to manage listings and users, including a multi-tiered ad approval system (manual/automatic based on user type/ad type).
